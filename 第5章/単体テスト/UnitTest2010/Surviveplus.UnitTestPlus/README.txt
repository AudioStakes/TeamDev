Surviveplus.UnitTestPlus.0.0.5-bata1.nupkg

このファイルは公開されていない NuGet パッケージファイルです。
公開されている Surviveplus.UnitTestPlus は、.NET Framework 4.5 用のパッケージですが、
こちらは、Visual Studio 2010 でも使用できるように .NET Framework 4.0 用のパッケージに変更してあります。

Visual Studio 2010 のツールメニューの［NuGet パッケージマネージャ］＞［パッケージマネージャの設定］を変更して、
このファイルをおいたフォルダを［パッケージソース］に追加するとインストールすることができます。

Visual Studio 2012 以降で使用する場合は、NuGetパッケージマネージャで最新のパッケージをダウンロードしてインストールしてください。
